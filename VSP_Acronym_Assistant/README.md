# VSP Acronym Assistant

Author: Janet Portillo (Jport-GH)
Course: CSCI 1151 – Python Programming  
Project: Chapters 1–7 Final Project  

## The VSP Acronym Assistant is a command-line tool for CMF SBU to quickly look up, manage, and understand acronyms used across VSP’s units, systems, membership, financial, and benefit terms.

## Planned Features:
- Search for an acronym
- Add new acronyms
- Edit or delete acronyms
- List acronyms by category
- Save and load acronym data using JSON
- Use a simple menu-driven interface
